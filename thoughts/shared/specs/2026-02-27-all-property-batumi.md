# All Property Batumi (apb.ge) — Specification

## Executive Summary

All Property Batumi (apb.ge) — сайт-каталог недвижимости в Батуми, позиционируемый как агрегатор-площадка. Фактически это маркетинговый инструмент одного агентства, замаскированный под нейтральный маркетплейс. Цель: привлечь международную аудиторию (RU, EN, AR, TR, KA) через SEO, вызвать доверие через позицию "площадки", конвертировать посетителей в лиды.

## Problem Statement

### Текущая ситуация
- Информация о недвижимости в Батуми разрозненная: десятки сайтов застройщиков, соцсети, мессенджеры
- Покупатели (особенно иностранные) не доверяют отдельным агентствам — предпочитают "агрегаторы"
- Нет единой площадки с качественной фильтрацией, мультиязычностью и удобным UX для рынка Батуми

### Решение
Создать сайт, который **выглядит как агрегатор** (множество объектов, разные "агенты", нейтральный бренд площадки), но все лиды идут одному агентству. Тройная выгода: доверие + SEO-масштаб + контроль над воронкой.

## User Personas

### 1. Русскоязычный инвестор (основной)
- **Профиль:** 30-55 лет, Россия/СНГ, средний+ доход
- **Цель:** Купить квартиру для сдачи в аренду или переезда
- **Поведение:** Ищет "купить квартиру Батуми", сравнивает ЖК, считает доходность
- **Язык:** Русский

### 2. Арабский инвестор
- **Профиль:** 35-60 лет, ОАЭ/Саудовская Аравия/Кувейт, высокий доход
- **Цель:** Инвестиционная недвижимость, диверсификация портфеля
- **Поведение:** Ищет "عقارات باتومي", важна визуальная подача и доходность
- **Язык:** Арабский (RTL!)

### 3. Турецкий покупатель
- **Профиль:** 30-50 лет, Турция, средний доход
- **Цель:** Покупка для жизни или аренды (географическая близость)
- **Поведение:** Ищет "Batum emlak", "Batum daire fiyatları"
- **Язык:** Турецкий

### 4. Международный покупатель (EN)
- **Профиль:** 30-55 лет, Европа/США/другие, средний+ доход
- **Цель:** Инвестиция или lifestyle relocation
- **Поведение:** Ищет "buy property Batumi Georgia", "Batumi real estate investment"

### 5. Грузинский покупатель
- **Профиль:** Местный рынок, покупка для жизни
- **Язык:** Грузинский

## User Journey

### Первичный визит (30 секунд)
1. Пользователь попадает на главную → видит **красивую сетку объектов** с фото и ценами
2. Замечает **фильтры** (тип, район, цена, площадь) — ощущение большой площадки
3. Прокручивает вниз → видит **интерактивную карту Батуми** с метками объектов
4. Замечает **разных агентов** у разных объектов — выглядит как маркетплейс

### Просмотр объекта
1. Клик на карточку → страница объекта с галереей фото, описанием, характеристиками
2. Видит **агента** (фото + имя + кнопка связи) — создаёт ощущение персонального подхода
3. Может **скачать PDF** с описанием объекта
4. Может добавить в **избранное** или **сравнение**

### Конверсия
1. Нажимает "Связаться с агентом" → форма заявки или WhatsApp/Telegram
2. Заявка уходит агентству → менеджер связывается
3. Пользователь не знает, что все "агенты" — сотрудники одной компании

### Дополнительный контент
1. Читает блог: "Как иностранцу купить недвижимость в Грузии"
2. Смотрит страницу ЖК: фото, планировки, отзывы, ход строительства
3. Изучает гайд по районам: где лучше жить/инвестировать в Батуми

## Technical Architecture

### Tech Stack

#### Frontend
- **Next.js 15** (App Router) — SSR/SSG, оптимальный для SEO
- **TypeScript** — типизация
- **Tailwind CSS v4** — стилизация, RTL-поддержка через `dir="rtl"` + logical properties
- **React Query (TanStack Query)** — серверное состояние
- **next-intl** — мультиязычность с URL-префиксами
- **Leaflet + OpenStreetMap** — интерактивная карта (бесплатно, open-source)
- **Puppeteer** — серверная генерация PDF (красивый, брендированный)

#### Backend / CMS
- **Sanity CMS** — headless CMS с визуальным редактором
  - Простой UI для нетехнической команды
  - Встроенная поддержка мультиязычного контента
  - Real-time collaboration
  - Image pipeline (ресайз, кроп, CDN)
  - GROQ для гибких запросов
- **Next.js API Routes** — для форм заявок, генерации PDF, отправки email

#### Hosting & Infrastructure
- **Vercel** — хостинг Next.js (Edge Network, автоматический CDN)
- **Sanity Cloud** — хостинг CMS (бесплатный tier покрывает 50-200 объектов)
- **Cloudflare** — DNS, дополнительный CDN для изображений
- **Resend / SendGrid** — отправка email-уведомлений о заявках

## Appendix: Research Findings

### Референс: ЦИАН (cian.ru)
- Чистый, структурированный дизайн без визуального шума
- Мощные фильтры с мгновенным обновлением
- Карта с маркерами объектов
- Карточки агентов на каждом объекте
- **Что взять:** структуру каталога, фильтры, чистоту UI
- **Что не брать:** перегруженность мобильной версии

### Антиреференс: Avito Недвижимость
- Слишком много графических элементов
- Визуальный шум, перегрузка информацией
- **Вывод:** наш дизайн должен быть чище и спокойнее

### Конкурентный ландшафт Батуми
- Нет сильного мультиязычного агрегатора
- Большинство сайтов — одноязычные (RU или EN)
- Арабский и турецкий рынки практически не покрыты
- **Возможность:** стать первым качественным мультиязычным агрегатором
